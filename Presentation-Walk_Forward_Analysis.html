Walk-Forward Analysis

Summary

  • Walk-Forward Analysis tests whether a trading strategy's performance remains consistent when applied to new, unseen data.

  • It simulates how a trader would actually use the strategy: optimize on past data, trade on future data, then repeat.

Core Concept

  • Simulating real-world Use: WFA validates trading strategies by simulating real-world deployment through sequential train-test periods.

  • Process: The process divides historical data into rolling windows—training on period t to t+n, testing on t+n to t+n+m, then advancing by m periods and repeating—to assess whether in-sample performance persists out-of-sample.

  • Measuring Performance: Performance is typically measured using the Sharpe ratio

S = (r̄ - rf) / σr

where r̄ is mean return, rf is risk-free rate, and σr is the standard deviation of the return.

Deterioration is calculated as:

D = (# windows where S_test < S_train) / (Total # windows) × 100%

A deterioration rate exceeding 50% indicates potential overfitting, as the strategy fails to maintain its optimized performance on fresh data.

The SPX 0 DTE Iron Condor's 53.3% deterioration rate suggests its parameters capture historical noise rather than persistent market inefficiencies, making future performance unreliable.

Average Profit/Trade for a Window
Average Profit = (1/n) × Σ(P/Li)

Where:
- n = number of trades in the window
- P/Li = Profit/Loss of trade i
- Σ = sum from i=1 to n

Window 3 Training (trades 41-140, n=100):
P/L values include: [+2000, -1500, +3000, -500, +1000, ...] summing to $87,500
Average Profit = (1/100) × 87,500 = $875

Window 3 Testing (trades 141-160, n=20):
P/L values include: [-2000, +500, -1000, +300, ...] summing to -$4,000
Average Profit = (1/20) × (-4,000) = -$200

Win Rate for a Window
Win Rate = (Number of Winning Trades / Total Number of Trades) × 100%

Where:
- Winning Trade = any trade where P/L > 0
- Win Rate = (Σ(1 if P/Li > 0, 0 otherwise) / n) × 100%

Window 3 Training:
65 trades with P/L > 0 out of 100 total trades
Win Rate = (65/100) × 100% = 65%

Window 3 Testing:
8 trades with P/L > 0 out of 20 total trades
Win Rate = (8/20) × 100% = 40%

Standard Deviation (Volatility) for a Window
Standard Deviation (σ) = √[(1/n) × Σ(P/Li - μ)²]

Where:
- μ = Average Profit (mean) = $875 for training
- P/Li = Profit/Loss of trade i
- n = number of trades

Window 3 Training:
Deviations: [(2000-875)², (-1500-875)², (3000-875)², (-500-875)², ...]
= [1,265,625, 5,640,625, 4,515,625, 1,890,625, ...]
Sum of squared deviations = 306,250,000
Variance = 306,250,000/100 = 3,062,500
σ = √3,062,500 = $1,750

Window 3 Testing (μ = -$200):
Deviations: [(-2000-(-200))², (500-(-200))², (-1000-(-200))², ...]
= [3,240,000, 490,000, 640,000, ...]
Sum of squared deviations = 24,200,000
Variance = 24,200,000/20 = 1,210,000
σ = √1,210,000 = $1,100

Sharpe Ratio or a Window
Sharpe Ratio = (μ - Rf) / σ

Where:
- μ = Average Profit per Trade
- Rf = Risk-free rate (typically annualized, needs adjustment for trade frequency)
- σ = Standard Deviation of Profits

Assuming Rf = 5% annually, with ~250 trading days, ~1 trade/day:
Daily Rf = 0.05/250 = 0.0002 = 0.02% per trade
Dollar Rf per trade (on $10,000 capital) = $10,000 × 0.0002 = $2

Window 3 Training:
Sharpe = (875 - 2) / 1,750 = 873 / 1,750 = 0.499

Window 3 Testing:
Sharpe = (-200 - 2) / 1,100 = -202 / 1,100 = -0.184

Note: Simplified version (Rf = 0) gives 0.50 and -0.18 respectively

Deterioration for a Window
Deterioration(metric) = {
    1, if metric_testing < metric_training
    0, if metric_testing ≥ metric_training
}

Window 3 Results:
Sharpe Deterioration: 0.499 → -0.184, so -0.184 < 0.499 → Deterioration = 1
Win Rate Deterioration: 65% → 40%, so 40% < 65% → Deterioration = 1
Profit Deterioration: $875 → -$200, so -$200 < $875 → Deterioration = 1

Additional Quality Metrics
1. Consistency of Deterioration
   - Is it always slightly worse? (OK)
   - Or wildly inconsistent? (Bad)

2. Magnitude of Deterioration
   - Sharpe 1.0 → 0.9 = Small deterioration (10%)
   - Sharpe 1.0 → -0.5 = Massive deterioration (150%)

3. Profit Sign Changes
   - Training: +$1000 average → Testing: +$800 (OK, just less profit)
   - Training: +$1000 average → Testing: -$500 (BAD, losing money!)

Good Versus Bad Scenarios
GREEN FLAGS (Valid Strategy):
✓ Deterioration Rate < 40%
✓ Consistent positive Sharpe in testing periods
✓ No profit sign reversals (profitable stays profitable)
✓ Testing Sharpe > 0.5 in most windows
✓ Win rate stays above 45% in testing

RED FLAGS (Invalid/Overfitted Strategy):
✗ Deterioration Rate > 50% (like yours at 53.3%)
✗ Multiple windows with negative testing Sharpe
✗ Profit reverses from positive to negative
✗ Wild swings in performance between windows
✗ Testing performance consistently 50%+ worse than training

Professional Trading Standards
Typical Institutional Requirements (Hedge Funds/Prop Firms):
- Maximum acceptable deterioration: 30-40%
- Minimum testing Sharpe: 0.5-1.0
- Must be profitable in 70%+ of testing windows
- No single testing window loss > 2x average training profit
